# 📋 Board Master - Sistema Avançado de Gestão de Tarefas

## 📌 Descrição
Sistema completo de gerenciamento de tarefas estilo **Kanban** com múltiplos boards, colunas, cards, usuários, labels, comentários, anexos, datas de vencimento e muito mais. Desenvolvido em Java puro com JDBC, H2, Flyway e padrões de projeto.

## 🛠️ Tecnologias
- Java 11
- Maven
- H2 Database
- Flyway (migrações)
- SLF4J (logging)
- JUnit 5 & Mockito (testes)
